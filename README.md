# python-package-demo

Demo Python package for the COSI meeting on 2025-02-05

Quick instructions:

### 1. Create a virtual environment and activate it.
```
$ python3 -m venv .venv
$ source .venv/bin/activate
```

### 2. Install dependencies.
```
$ pip install -r requirements.txt
```

### 3. Test the package.
```
$ make test
```

### 4. Build the package.
```
$ make build
```
